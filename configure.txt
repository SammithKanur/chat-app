
Install docker

Install turnServer
1. refer this https://ourcodeworld.com/articles/read/1175/how-to-create-and-configure-your-own-stun-turn-server-with-coturn-in-ubuntu-18-04
2. change its config in /etc/turnserver.conf with the one in project
3. restart turn server

ElasticSearch
1. run mkdir elasticsearch
2. grant permissions to it.

1. Follow steps here https://certbot.eff.org/lets-encrypt/ubuntufocal-webproduct
2. Once certbot is installed, use its help command to learn more about it.
